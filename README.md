## 💻 Aplicação

Foi desenvolvida como site e portfolio pessoal utilizando as tecnologias listadas abaixo. O layout foi baseado em um modelo gratuito do _jamstackthemes.dev_.

O site contém:

-   Formação
-   Sobre
-   Postagens (markdown)
-   Links de Redes Sociais

## 🚀 Tecnologias
